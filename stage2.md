# 阶段 2

> 我只需要弄清楚事情是如何运作的。——伊万·萨瑟兰

我们已经领略了很多图形学的侧面，希望它正如你想象的那样有趣或令人惊叹，如果超乎你的想象就再好不过了！

在这个阶段，我们希望你深入到真正的计算机图形学，触摸她的灵魂——而不是仅仅关注到*外表*。

> *外表*：指各种具体的引擎工具、图形 API 等。

这就好像计算机科学和编程语言的关系，如果仅停留在后者而忽略前者，那么你就会沦为 **Code Monkey** 而不是成为 **Craftsman** ——尤其是在 Agentic Coding 越发强悍的时代，对基础的理解已经比以往任何一个时代都更重要。

你仍然需要进行大量图形编程练习，但它们不再是学习的唯一重心——你会投入比以往更多的精力去建立各种系统、算法和技术的心智模型。

好了，多说无益，来看看你即将开启的新旅程吧！
